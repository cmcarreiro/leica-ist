Detailed Description

This is the main factory abstract class: it provides methods for creating the lexical analyser and the compiler itself. Instances of concrete subclasses will be created by the "main" function to provide instances of the scanner and compiler objects for a concrete language.

  • keys are language names
  • values are compiler factories
